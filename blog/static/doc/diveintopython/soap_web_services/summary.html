12.9. 小结

SOAP 网络服务是很复杂的，雄心勃勃的它试图涵盖网络服务的很多不同应用。这一章我们接触了它的一个简单应用。

在开始下一章的学习之前，确保你能自如地做如下工作：

  • 连接到 SOAP 服务器并调用远程方法
  • 通过 WSDL 文件自省远程方法
  • 有效排除 SOAP 调用中的错误
  • 排除常见的 SOAP 相关错误
